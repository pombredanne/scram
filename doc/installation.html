Installation Instructions¶

Note

Some SCRAM tools and tests may be absent in distribution packages.

Ubuntu 14.04 or later¶

Run the following commands to get SCRAM from its PPA.

sudo add-apt-repository ppa:rakhimov/scram

sudo apt-get update

sudo apt-get install scram

Other Platforms¶

Please follow the building and installing instructions on GitHub.
