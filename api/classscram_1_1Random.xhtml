Detailed Description

This class contains generators for various random distributions.

The values passed to the member functions are asserted to be in the correct form. In other words, the user should make sure that the passed parameters are valid. For example, standard deviation cannot be negative.

This facility wraps the engine and distributions. It provides convenience and reproducibility for the whole analysis.
