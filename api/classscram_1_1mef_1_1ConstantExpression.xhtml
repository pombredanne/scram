bool scram::mef::ConstantExpression::IsConstant ( )
inlineoverridevirtualnoexcept

Determines if the value of the expression varies.

The default logic is to check arguments with uncertainties for sampling. Derived expression classes must decide if they don't have arguments, or if they are random deviates.

Returns
true if the expression's value does not need sampling.
false if the expression's value has uncertainties.
Warning
Improper registration of arguments may yield silent failure.
