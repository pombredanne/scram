Detailed Description

Boolean variables in a Boolean formula or graph.

Variables can represent the basic events of fault trees.

Indexation of the variables is special. It starts from 1 and ends with the number of the basic events in the fault tree. This indexation technique helps preprocessing and analysis algorithms optimize their work with basic events.
