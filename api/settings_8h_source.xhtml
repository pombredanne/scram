29 /// Builder for analysis settings.
30 /// Analysis facilities are guaranteed not to throw or fail
31 /// with an instance of this class.
32 ///
33 /// @warning Some settings with defaults and constraints
34 /// may have side-effects on other settings.

36 /// @warning The order of building the settings matters.

42  /// Sets the algorithm for Qualitative analysis.
43  /// Appropriate defaults are given to other settings
44  /// relevant to the algorithm.
45  ///
46  /// MOCUS and ZBDD based analyses run
47  /// with the Rare-Event approximation by default.
48  /// Whereas, BDD based analyses run with exact quantitative analysis.
49  ///
50  /// @param[in] algorithm The name of the algorithm in lower case.

177  /// Sets the flag for probability analysis.
178  /// If another analysis requires probability analysis,
179  /// it won't be possible to turn off probability analysis

181  ///
